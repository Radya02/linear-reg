#  Energy Consumption Prediction using Linear Regression

##  Описание проекта

**Energy Consumption Prediction** — это проект машинного обучения, предназначенный для прогнозирования потребления энергии зданием на основе его характеристик.

Модель обучается на исторических данных и затем используется для предсказания будущего энергопотребления на основе введённых пользователем параметров.

Проект состоит из двух основных частей:

1. **Обучение модели (training)**
2. **Предсказание новых данных (prediction)**

---

# Используемые библиотеки

Проект реализован на языке **Python** с использованием следующих библиотек:

* **pandas** — обработка табличных данных
* **numpy** — работа с массивами
* **scikit-learn** — машинное обучение
* **matplotlib** — визуализация данных
* **joblib** — сохранение и загрузка модели

---

#  Данные

Модель обучается на данных о зданиях.

## Пример структуры данных:

| Feature            | Описание                                         |
| ------------------ | ------------------------------------------------ |
| Building Type      | Тип здания (Residential, Commercial, Industrial) |
| Square Footage     | Площадь здания                                   |
| Occupants          | Количество жителей                               |
| Appliances         | Количество приборов                              |
| Temperature        | Средняя температура                              |
| Energy Consumption | Потребление энергии (целевая переменная)         |
